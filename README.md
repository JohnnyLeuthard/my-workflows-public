# my-workflows-public

A collection of workflow structures and references for organizing AI-assisted work (primarily with Claude Code). I use Visual Studio Code (VSC) and Claude plugin. The CLI needs to be installed.  

This repo is itself an MWP (Model Workspace Protocol) workspace. Entry points are [CLAUDE.md](CLAUDE.md) and [CONTEXT.md](CONTEXT.md) at this level — Claude reads those first and routes from there.

## Contents

| Folder | What it is |
|--------|------------|
| [CyberArk/](CyberArk/) | My own active workspace — CyberArk vault automation using the Model Workspace Protocol (MWP). |
| [corp-brand/](corp-brand/) | Reusable public-safe brand kit for writing, color, icon, and logo guidance used by other workflows. |
| [clief-notes/](clief-notes/) | **Reference only.** Example workspaces and tutorials from the Clief-Notes Skool community (`The Foundation/`, `Folder Structure/`). See [clief-notes/README.md](clief-notes/README.md). |
