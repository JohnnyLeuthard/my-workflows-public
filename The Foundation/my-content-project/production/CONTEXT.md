# Production

This workspace manages the production pipeline from brief to finished output.

## Workflow
1. **Briefs** — Production briefs defining scope, requirements, and goals
2. **Specs** — Technical specifications and asset requirements
3. **Builds** — Work-in-progress production files
4. **Output** — Finished, export-ready content

## Guidelines
- Every production piece starts with a brief in /briefs
- Specs should reference their corresponding brief
- Final output goes in /output before handoff to /distribution
