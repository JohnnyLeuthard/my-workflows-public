# Script Lab

This is the creative workspace for ideation, drafting, and finalizing content scripts.

## Workflow
1. **Ideas** — Raw concepts, brainstorms, and pitches go in /ideas
2. **Drafts** — Work-in-progress scripts go in /drafts
3. **Final** — Approved, polished scripts go in /final

## Guidelines
- Each idea should be its own file with a descriptive name
- Drafts should reference which idea they originated from
- Only move to /final when the script is ready for production
