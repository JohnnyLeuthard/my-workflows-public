# iScaleLabs — Workspace Identity

This workspace holds iScaleLabs projects: agent-control experiments, workflow prototypes, and tutorial follow-alongs.

## Protocol

- **Route first**: Read `CONTEXT.md` in this directory before opening any project folder.
- **Project isolation**: Each project under `iScaleLabs/` is self-contained. Do not cross-load between projects unless explicitly asked.
- **Follow AGENTS.md**: Projects here use `AGENTS.md` files to drive stage-by-stage execution. Follow them in sequence.
