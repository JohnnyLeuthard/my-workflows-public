## Overview

We are looking for a Senior DevOps Engineer to join our Platform Engineering team. In this role you will own the infrastructure that powers our production environment, drive reliability improvements, and mentor junior engineers on DevOps best practices. You will work closely with software engineering teams to embed DevOps culture throughout the development lifecycle.

---

## Key Responsibilities

- Design, build, and maintain CI/CD pipelines using GitHub Actions and Jenkins
- Own Kubernetes cluster administration (deployment, scaling, upgrades, RBAC)
- Manage infrastructure-as-code with Terraform and Helm across cloud environments
- Monitor system health using Prometheus, Grafana, and centralized logging
- Define and enforce SLOs/SLAs; lead incident response and post-mortems
- Partner with security team on vulnerability remediation and compliance controls
- Develop internal tooling to automate repetitive operational tasks
- Mentor junior DevOps engineers and conduct code/IaC reviews

---

## Required Skills

- Kubernetes (cluster administration, namespaces, RBAC, networking)
- Docker and container lifecycle management
- Terraform (modules, state management, workspaces)
- CI/CD: GitHub Actions and/or Jenkins
- Linux (RHEL/Ubuntu administration, shell scripting)
- Python or Bash for automation scripting
- Cloud platforms: AWS, Azure, or GCP (any one)

---

## Nice-to-Have Skills

- CKA (Certified Kubernetes Administrator) or CKAD
- Helm chart authoring
- Ansible or similar configuration management
- Service mesh experience (Istio, Linkerd)
- AWS Solutions Architect or equivalent cloud certification
