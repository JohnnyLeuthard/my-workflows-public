## Overview

We are seeking a QA Engineer on a 12-month contract engagement to expand our automation testing capabilities. You will work alongside the development team to design, implement, and maintain automated test suites for our web applications and APIs. This role is ideal for someone who thrives in an Agile environment and can independently deliver high-quality test frameworks.

---

## Key Responsibilities

- Design and implement automated test suites using Selenium WebDriver and Python
- Build and maintain API test suites using Postman / REST Assured
- Integrate tests into CI/CD pipelines (GitHub Actions)
- Write and maintain test plans, test cases, and defect reports in JIRA
- Conduct regression, smoke, and exploratory testing across environments
- Participate in sprint ceremonies and contribute to Definition of Done criteria
- Identify gaps in test coverage and proactively address them

---

## Required Skills

- Selenium WebDriver with Python or Java
- API testing: Postman and/or REST Assured
- JIRA for test management and defect tracking
- Agile / Scrum methodology
- Git for version control

---

## Nice-to-Have Skills

- Performance testing: JMeter or k6
- Mobile testing: Appium
- BDD: Cucumber / Behave
- CI/CD integration experience (GitHub Actions preferred)

- **Education**: BS in Information Systems, Computer Science, or equivalent
