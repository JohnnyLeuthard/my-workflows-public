# CCP Stage 01 Router: Config Model

## Use This Stage For

- Defining target entries for VIPs, LTMs, GSLB endpoints, and direct CCP hosts.
- Capturing approved ports per environment and target.
- Recording TLS/certificate validation notes and ownership metadata.
- Producing `output/config_model.md`.

## Required References

Read `../../references/_INDEX.md`, then `../../references/config_model.md`.

## Stop Rule

After producing or updating the config model artifact, stop and wait for human approval before moving to request cataloging.
