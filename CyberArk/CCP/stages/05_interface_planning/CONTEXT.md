# CCP Stage 05 Router: Interface Planning

## Use This Stage For

- Planning dropdowns, selectors, forms, and validation.
- Evaluating deployment options under internal security restrictions.
- Separating read-only URI building from approved live testing.
- Producing `output/interface_plan.md`.

## Required Inputs

Use the latest approved artifacts from earlier stages when available.

## Stop Rule

After producing or updating the interface plan, stop and wait for human review before implementation work.
