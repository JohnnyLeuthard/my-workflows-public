# CCP Stage 03 Router: URI Builder

## Use This Stage For

- Combining approved target config and request items into request URI documentation.
- Checking that URI inputs are complete enough for a future endpoint test.
- Producing `output/uri_set.md`.

## Required Inputs

- `../01_config_model/output/config_model.md`
- `../02_request_catalog/output/request_catalog.md`

## Stop Rule

After producing or updating the URI set artifact, stop and wait for explicit approval before endpoint testing.
