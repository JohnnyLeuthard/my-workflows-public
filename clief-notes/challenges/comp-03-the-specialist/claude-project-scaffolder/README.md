# Claude Project Scaffolder

Drop this folder into any Claude project. Describe what specialist or expert you want to build in plain language — what problem it solves, who'd use it, what it's great at. The scaffolder designs a complete folder structure using Interpretable Context Methodology (ICM), including the exact files you need, what goes in each one, what source material to gather, and the opening prompt to activate the context in Claude. It teaches you the pattern as it helps you build.

**Example:** "I want to build a specialist who critiques data visualizations — spots mistakes in axis ranges, wrong chart types, color accessibility issues." → Get back a complete folder structure with file purposes, reference material to gather, and a ready-to-paste opening prompt.
