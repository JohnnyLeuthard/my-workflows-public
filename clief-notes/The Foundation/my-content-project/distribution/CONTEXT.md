# Distribution

This workspace manages content distribution, scheduling, and performance tracking.

## Workflow
1. **Platforms** — Platform-specific configurations and requirements
2. **Scheduling** — Content calendars and publishing schedules
3. **Analytics** — Performance data and insights

## Guidelines
- Each platform should have its own config file in /platforms
- Scheduling files should reference the content from /production/output
- Analytics should be organized by time period or campaign
