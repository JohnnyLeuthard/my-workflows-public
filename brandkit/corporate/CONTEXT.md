# Brand Kit Router

Read the user's request and load only the brand file that applies.

## Routing Rules

| If the request involves... | Load... |
|---|---|
| Brand purpose, audience, personality, promise, positioning | [01-primary-brand.md](01-primary-brand.md) |
| Voice, tone, emotional feel, communication style | [02-voice-and-tone.md](02-voice-and-tone.md) |
| Grammar, formatting, terminology, content patterns | [03-writing-guide.md](03-writing-guide.md) |
| Colors, contrast, UI palettes, visual mood | [04-colors.md](04-colors.md) |
| Icons, pictograms, UI symbols, icon accessibility | [05-icons.md](05-icons.md) |
| Logo, wordmark, lockups, clear space, logo misuse | [06-logo.md](06-logo.md) |
| General setup or folder overview | [README.md](README.md) |

## Loading Rules

- Do not load every file by default.
- Load the smallest useful file for the task.
- Keep this public starter kit free of private company information.
