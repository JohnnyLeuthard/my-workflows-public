# Brand Setup — Guided Questionnaire

Capture all brand decisions here first. The answers populate the brand files (01–06).

**Two ways to use this:**
- **AI-guided**: Say "set up my brand" and the AI reads `CLAUDE.md`, asks each question, and fills in your answers automatically.
- **Manual**: Fill in each answer directly, then edit the `{{PLACEHOLDER}}` tokens in each brand file yourself.
