# Brand Kit Workspace

You are in the `corp-brand/` workspace. This workspace provides reusable brand, writing, color, icon, and logo guidance for other workflows in this repository.

## Purpose

Use these files as a neutral starter brand kit. They should help AI tools produce consistent public-facing content without relying on any private company, client, or proprietary brand system.

## Rules

- Treat this folder as reusable reference material, not as a finished real-world brand.
- Do not add company-specific names, private assets, internal system names, or proprietary standards.
- Keep guidance practical and editable.
- Prefer short markdown sections, tables, and decision lists that are easy for humans and AI tools to scan.
- If a downstream workflow needs brand or writing context, load only the specific brand file needed for that task.
